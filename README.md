## 2. Elite Availability Scheduler
A premium, interactive 6-day availability table designed for seamless scheduling. Features a highly visual way to manage time slots across a standard work week.

### Features
- **Interactive Level Selection:** Dynamic selection of availability states (Available, Busy, Tentative).
- **Persistent Logic:** Utilizes `localStorage` to ensure data persists across browser reloads.
- **Glassmorphism UI:** Semi-transparent aesthetic with vibrant gradients.
- **Dynamic Slot Generation:** Automated generation of hourly slots.

### Technologies Used
- **CSS3:** Custom hard-stop linear gradients and backdrop-filter effects.
- **JavaScript:** Handles interactive state and local storage management.
